# Shopify Functions - Product Page Functionality

## Опис

Повнофункціональна сторінка продукту з різноманітними інтерактивними компонентами: бігучий текст з відгуками, FAQ акордеон, карусель продуктів та відео-карусель з відгуками.

## Основні компоненти

### 1. Header Ticker (Бігучий текст з відгуками)
**js:**: `scripts/ticker_shopify.js`
**styles**: `styles/header.css`

- Безперервна прокрутка текстових відгуків у шапці сайту
- Налаштування швидкості через `data-shopify-ticker-speed`
- Підтримка зворотного напрямку через `data-shopify-ticker-is-reversed`
- Автоматична реплікація контенту для безшовної прокрутки
- Підтримка паузи при наведенні через `data-shopify-ticker-pause`
- Адаптивна робота через ResizeObserver
- Є 2 блоки `shopify-ticker`. Перший - куди потрібно розміщувати текстовий контент. Другий - куди js код вставить необхідну кількість клонів контенту з першого блоку.


### 2. FAQ Accordion (Акордеон питань та відповідей)
**js:**: `scripts/accordion.js`
**styles**: `styles/accordion.css`

- Анімоване розкриття/згортання відповідей
- Автоматичне закриття інших питань при відкритті нового (accordion-поведінка)
- Зміна іконки з "+" на "×" при розкритті
- Плавні CSS-транзиції з автоматичною висотою
- Підтримка ARIA-атрибутів для доступності
- Адаптивність при зміні розміру вікна

- За замочуванням зображення для іконки береться з `scripts/accordion.js`
- Якщо в блоці `accordion-container` є клас `rotate-icon`, то js буде додавати стилі для повороту іконки + і робити з неї х
  - В цьому режимі використовується лише зображення прописане в html!
- Якщо в блоці accordion-container є клас `accordion-open-first`, то js зробить перший елемент відкритим за замовчуванням. Не треба намагатись міняти стилі для першого елементу вручну!
- Структура:
"accordion-item"
	"accordion-question"
		"accordion-icon"
	"accordion-answer"


### 3. Product Carousel (Карусель продуктів)
**Файл:** `scripts/product_carousel.js`

- Головний слайдер з великими зображеннями продукту
- Синхронізована карусель мініатюр знизу
- Завантаження даних з JSON файлу (`data/carousel.json`)
- Інтеграція зі Swiper.js для плавної навігації
- Підтримка srcset для адаптивних зображень
- Lazy loading для оптимізації продуктивності
- Навігація кнопками "попередній" / "наступний"

### 4. Video Carousel (Відео-карусель з відгуками)
**Файли:** `scripts/slider.js`, `scripts/video.js`

#### Перетягування на десктопі
- Підтримка перетягування мишею (drag & drop) для прокрутки каруселі
- Автоматичне визначення різниці між кліком та перетягуванням
- Курсор змінюється на `grab` при наведенні та `grabbing` під час перетягування

#### Оптимізація продуктивності
- Використання `requestAnimationFrame` для плавної прокрутки
- Оптимізація оновлення кнопок навігації (тільки після завершення перетягування)

#### Запобігання випадковому запуску відео
- Відео не запускаються під час перетягування каруселі
- Розпізнавання перетягування на основі часу та руху миші

#### Автоматична зупинка відео
- При запуску нового відео всі інші відео в каруселі автоматично зупиняються
- Завжди відтворюється лише одне відео одночасно
- Кнопки play інших відео автоматично стають видимими

#### Управління відтворенням
- Клік на кнопку play або відео для запуску/паузи
- Автоматичне відображення кнопки play після завершення відео
- Підтримка poster-зображень для кожного відео

#### Навігація кнопками
- Кнопки "попередній" / "наступний" для прокрутки
- Автоматичне приховування кнопок на краях каруселі

## Структура файлів

### JavaScript
- `scripts/ticker_shopify.js` - бігучий текст з відгуками
- `scripts/accordion.js` - FAQ акордеон
- `scripts/product_carousel.js` - карусель продуктів з Swiper.js
- `scripts/slider.js` - логіка перетягування та навігації відео-каруселі
- `scripts/video.js` - управління відео та зупинка інших при запуску нового

### Styles
- `styles/reset.css` - базові стилі та скидання
- `styles/header.css` - стилі для ticket
- `styles/accordion.css` - стилі для FAQ секції
- `styles/sp-product-block.css` - стилі блоку продукту
- `styles/product-carousel.css` - стилі каруселі продуктів
- `styles/slider.css` - стилі відео-каруселі

### Дані
- `data/carousel.json` - дані для каруселі продуктів (зображення, мініатюри, конфігурація)

## Залежності

- **Swiper.js** - для каруселі продуктів (завантажується динамічно)
- **Google Fonts** - Onest, Piazzolla
- **Shopify CDN** - для зображень та відео

## Особливості реалізації

- Адаптивний дизайн (мобільна та десктопна версії)
- Оптимізація продуктивності (lazy loading, requestAnimationFrame)
- Доступність (ARIA-атрибути)
- Плавні анімації та переходи
- Синхронізація між компонентами (мініатюри та головний слайдер)
